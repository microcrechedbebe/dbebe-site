# D'BEBE - Site Web Micro-crèche

Site web moderne pour la Micro-crèche D'BEBE à Besançon.

## 🚀 Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (animations)
- **Lucide React** (icônes)

## 🎨 Design

- Design moderne et épuré
- Animations scroll-triggered
- 100% responsive (mobile-first)
- Palette bleu océan / corail / soleil

## 🏗️ Structure

- Hero animé avec vaguelettes marines
- Sections scroll avec animations
- Timeline interactive de la journée
- Formulaire d'inscription en 3 étapes
- Footer complet avec infos contact

## 📦 Installation

```bash
npm install
npm run dev
```

## 🚀 Déploiement

Déployé automatiquement sur Vercel via GitHub.
